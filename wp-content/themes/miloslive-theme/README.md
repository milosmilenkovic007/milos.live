# MilosLive Theme - Custom WordPress Theme

## Opis
MilosLive Theme je prilagođena WordPress tema razvijena bez nepotrebnih dodatnih pluginova.  
Optimizovana je za brzinu, SEO, i lako prilagođavanje.
